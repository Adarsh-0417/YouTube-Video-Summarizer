# 🎬 YouTube Video Summarizer (GenAI Project)

A simple yet powerful GenAI application that extracts transcripts from YouTube videos and generates concise summaries using transformer-based models.

---

## 🚀 Features

* 🔗 Input any YouTube video URL
* 📝 Automatically fetches video transcript
* 🤖 Generates summary using a pretrained transformer model
* ⚡ Chunk-based processing for long videos
* 🌐 Interactive UI using Gradio

---

## 🧠 Tech Stack

* **Python**
* **Transformers (HuggingFace)**
* **PyTorch**
* **Gradio**
* **YouTube Transcript API**

---

## ⚙️ How It Works

1. User inputs a YouTube URL
2. Video ID is extracted using regex
3. Transcript is fetched via YouTube Transcript API
4. Transcript is converted into plain text
5. Text is split into smaller chunks
6. Each chunk is summarized using a transformer model
7. Final summary is generated by combining chunk summaries

## 🖥️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/youtube-video-summarizer.git
cd youtube-video-summarizer
```

---

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate
```

---

### 3. Install dependencies

```bash
pip install torch transformers gradio youtube-transcript-api
```

---

## ▶️ Run the Application

```bash
python app.py
```

Then open:

```
http://127.0.0.1:7860
```

---

## ⚡ GPU Acceleration (Optional)

For faster performance, install CUDA-enabled PyTorch:

```bash
pip uninstall torch -y
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

⚠️ Requires Python 3.10/3.11 and compatible GPU.
